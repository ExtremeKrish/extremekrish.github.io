Host Your Website Using Github Pages

1. At First Go to Github.com and click on Sign Up button

2. Enter Your Email And Password and Make your Username

Tips: Choose a nice username that defines your site because it will be the name of your website.

3. Verify Your Email Address

4. After Verificatin Close All tabs And again Go to Github.com.
Then Click on "Create Repository" Button.

5. Name Your Repository in the form <Your Username>.github.io
This will be the url of your website.
Now when done Click on "Create Repository" in the bottom.

Tips: See the Screenshots If you could not understand

6. After that new page will open.
Click on "Create a New File" (See in Screenshot below)

7. Name the file as "index.html" and Type your HTML below

8. Click on Preview tab to see preview and click the "Commit New File Button" to save the file(See in Screenshot below)

9. After Your file is saved you see a page like below.
You can click on "View Code" to see all your saved files.

Note: Github Pages can be used to Host only static websites. Dynamic php website will not work with github pages. You can upload or create images, audio, video, html, css, js and many more.

10. Now Scroll the upper tabs horizontally and click on Settings (See in Screenshot)

11. Then Settings tab will open.
Scroll the tab to bottom till you see "Github Pages" Section like in screenshot below.
Now Click on "Branch: master" button

12. Now Click on the "main" (For New Repositories)

13. Now Click on "Save" button

14. Reload the page two or three times till "Your Site is published at <Your website URL>" is not written in the the "Github Pages section in settings" (See in Screenshot below)

Important Note: Everytime You upload or create a file in your repository or edit a file, it will take some to update changes to your website.
To check Whether changes are saved or not, go to your repository settings and in the "Github Pages" if it is written "Your website is published" then your changes are saved.

15. Click on Link to open the website
